# YtToMp3
Descarga una lista de música de youtube

## Instalación

Usando [pip](https://pip.pypa.io/en/stable/) 

```bash
pip install youtube-search-python pytube os_sys
```

## Uso

```bash
python3 ./downloader.py 'TuListaAqui.txt' 'NombreDirectorioSalida'
```
